搭建Hexo博客并部署到GitHub

发表于2022-05-11|更新于2022-05-11|Hexo教程
|阅读量:

一、安装 Hexo

二、初始化 Hexo

新建文件夹 blog 在文件夹根目录打开终端输入 hexo init 进行初始化，之后的操作都是在该文件夹下进行

三、启动 Hexo 博客

四、新建博客

  • 在文件夹中打开 source -> _posts，找到刚才创建的文章，打开可以进行编辑

  • 重新生成：

    1. 打开终端输入 hexo clean 清理一下

    2. 重新生成

    3. 启动

五、部署到GitHub

  1. 在 GitHub 上新建一个仓库，名字为 github 账户名+.github.io，例如我的仓库名就是 liu-v0916.github.io

  2. 安装 hexo-deployer-git 插件

  3. 设置 _config.yml 文件

  4. 部署到远端

    浏览器上输入 liu-v0916.github.io 就可以访问啦

六、自定义 Hexo 主题

这里是设置一个叫 yilia 的主题

  1. 下载主题 themes/yilia 意思是把文件下载到 themes/yilia 文件夹下

  2. 配置主题

    打开 _config.yml 文件

  3. 清理

  4. 生成

  5. 部署
